# openrasp-autopwn

autopwn 是一款类似 metasploit 的POC测试框架。为了验证OpenRASP漏洞覆盖程度，我们开发了这样一款工具，用于对漏洞进行批量验证和测试

这个工具里所覆盖的漏洞，将会和OpenRASP的[CVE覆盖列表](https://rasp.baidu.com/doc/usage/cve.html)一一对应
